## 💡 Core Features

### Cycle Analysis ✨
- **Single-anchor mode**: Project cycles from one historical point
- **Convergence mode**: Detect confluence from multiple anchors
- **16 cycle presets**: Core, Extended, Advanced + Custom
- **Trading day adjustment**: Handle weekends/holidays
- **Reaction classification**: 6 types of price reactions
- **Confidence scoring**: Measure signal strength

### Backtesting Engine ✨
- **Historical validation**: Test cycles against past data
- **Hit rate calculation**: By cycle length and asset type
- **Comprehensive reporting**: Detailed results and statistics

## 💾 Database Structure

### 9 Core Tables
- **assets** - Market metadata
- **anchors** - Historical highs/lows
- **cycle_projections** - Calculated dates
- **backtest_runs** - Test sessions
- **backtest_results** - Individual results
- **confluence_zones** - Multi-anchor results
- **indicator_snapshots** - Technical readings
- **saved_analyses** - User sessions
- **app_config** - Settings
